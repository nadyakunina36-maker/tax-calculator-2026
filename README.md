# Налоговый калькулятор 2026 — сравнение режимов (УСН/ОСН/ПСН/АУСН)

Интерактивный калькулятор для сравнения налоговой нагрузки клиента по режимам УСН 6%, УСН 15%, ОСН, ПСН и АУСН на 2026 год. Учитывает льготы по УСН и таблицу потенциального дохода по патенту для Республики Бурятия (Закон РБ от 28.11.2023 №179-VII).

Однофайловое статическое приложение — HTML + CSS + JS без зависимостей и сборки.

## Запуск локально

Просто откройте `index.html` в браузере.

## Публикация через GitHub Pages

1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main`, папка `/ (root)`
4. Сохранить — сайт появится на `https://<ваш-логин>.github.io/<название-репозитория>/`

## Источники ставок

- НК РФ (действующая редакция)
- ФЗ №425-ФЗ от 28.11.2025 (НДС 22% с 2026 г.)
- ФЗ №176-ФЗ от 12.07.2024 (налог на прибыль 25%)
- Закон Республики Бурятия от 28.11.2023 №179-VII (льготы по УСН и ПВД по патенту), с изм. от 29.04.2026 №1449-VII

## Дисклеймер

Расчёт справочный, не заменяет подготовку деклараций и налоговую консультацию. Актуальность ставок нужно сверять на дату использования.
